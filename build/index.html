Automatically discover, provision, and manage the resources to build virtual clusters—dynamic infrastructures that enable extreme scale scientific computation.

Discovery

Create an environment that facilitates the discovery of computing resources like processor time, data storage and networking across geographically distributed, dynamic virtual organizations.

Provisioning

Build an autonomous software agent that can provision resources with general guidance from scientists. Provisioning will be flexible, resilient, high throughput, and rapidly reconfigurable at runtime.

Management

Find ways to streamline data management and analysis. Make research results friendly to interactive analytics and discoverable by current and future team members.
